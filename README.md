# Troll Mario Game

Welcome to the Troll Mario Game!

This project is a 2D Unity game where players navigate through various levels filled with challenges, twists, and of course, trolls! 

## Getting Started

1. Clone the repository.
2. Open the project in Unity.
3. Enjoy the game!

## Project Structure
- **Assets/** - Contains all game assets
- **Scripts/** - Contains C# scripts for game mechanics
- **Scenes/** - Contains all game scenes
